## 2. Installation

Install **angular animations**
```bash
npm install @angular/animations
```

Install **talwindcss**

```bash
npm install tailwindcss
```

Install **tailjng** in your Angular project with npm

```bash
npm install tailjng
```

## 3. How to use
Verify **available components**

```bash
npx tailjng list
```
Install a component. In this case is a **button**.

```bash
npx tailjng add button
```
or

Install **all available** components
```bash
npx tailjng install-all
```
